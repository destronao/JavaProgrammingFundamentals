# Class Activities

Este directorio contiene las **actividades y prácticas realizadas siguiendo el orden del temario de clase**.

El objetivo principal es reforzar los conceptos vistos en cada unidad mediante ejercicios guiados y prácticos, manteniendo una estructura clara y coherente con la organización académica.

---

## Organización por temario

Cada tema del temario cuenta con su propio directorio y se divide de la misma forma que en clase:

- **activities**:  
  Ejercicios propuestos para practicar conceptos concretos del tema.
- **practices**:  
  Prácticas más completas o aplicadas relacionadas con el mismo contenido.

Esta separación permite diferenciar entre ejercicios de refuerzo y prácticas con mayor nivel de integración.

---

## Estructura general

Cada tema sigue la siguiente estructura:

```text
XX-topic-name/
├─ activities/
│ ├─ Exercise01.java
│ └─ README.md
├─ practices/
│ ├─ Practice01.java
│ └─ README.md
└─ README.md
```

- El **README del tema** incluye un breve resumen de:
  - conceptos utilizados
  - objetivos del tema
  - aspectos importantes a tener en cuenta al resolver los ejercicios

## Evolución futura

En el futuro se añadirá un **tercer tipo de paquete** destinado a la reconstrucción y mejora de actividades y prácticas ya realizadas.

Este nuevo espacio no estará limitado por el temario y tendrá como objetivo:
- Optimizar soluciones existentes
- Aplicar nuevos conocimientos a ejercicios antiguos
- Mejorar estructura, legibilidad y eficiencia del código

---

## Objetivo de este directorio

- Seguir de forma fiel el temario académico
- Consolidar las bases de programación en Java
- Mantener un histórico claro del progreso
- Facilitar la revisión y mejora de ejercicios a lo largo del tiempo
